Bubble Sort sorts its data by repeatedly comparing adjacent elements
It swaps them should they be in the wrong order
  • Step 1: In an unsorted array of 5 elements, start with the first two elements and sort them in ascending order. (Compare the element to check which one is greater)
  • Step 2: Compare the second and third element to check which one is greater, and sort them in ascending order
  • Step 3: Compare the third and fourth element to check which one is greater, and sort them in ascending order
  • Step 4: Compare the fourth and fifth element to check which one is greater, and sort them in ascending order
  • Step 5: Repeat steps 1â€“5 until no more swaps are required.
