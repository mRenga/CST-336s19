Merge Sort is an efficient, general-purpose, comparison-based sorting algorithm
It's a divide and conquer algorithm that was invented by John von Neumann in 1945
  • Step 1: Divide the unsorted list into N sublists, each containing 1 element
  • Step 2: Take adjacent pairs of two singleton lists and merge them to form a list of 2 elements. N will now convert into N/2 lists of size 2.
  • Step 3: Repeat the process till a single sorted list of obtained.
Merge Sort is just as easy as bubble sort to visualize, but the code for it might be challenging
